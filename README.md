# 🎾 Game Analytics: Unlocking Tennis Data

## 📌 Project Overview

This project is a Tennis Game Analytics Dashboard developed as part of the *Labmentix Data Analytics Internship*.

The application manages and analyzes tennis competition data using Python, SQLite, SQL, and Streamlit. It allows users to explore competitions, venues, competitors, and player rankings through an interactive dashboard.

## ▶️ How to Run the Project

### 1. Install Required Libraries


pip install -r requirements.txt


### 2. Generate Data


python generate_data.py


### 3. Load Data into SQLite


python data_loader.py


### 4. Launch the Dashboard


streamlit run app.py
